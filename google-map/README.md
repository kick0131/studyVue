# GoogleMapSample

---

### Project setup

```
npm install
```

### [IMPORTANT] add api key file

[src/constants/apikey.js]
```
export default Object.freeze({ 
  API_KEY: "XXXXX", 
})
```

### Compiles and hot-reloads for development

```
npm run serve
```

---
## install module

```
vue add vuetify
yarn add google-maps-api-loader
yarn add axios
```
