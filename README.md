# Sample projects

## Usage
### Create Project
1. run `vue cli`
    ```
    vue create my-project
    ```
2. install first setup
    ```
    cd my-project
    vue add vuetify
    yarn add vue-chartjs
    ```

### Execute Project
1. change into sample project directory.
    ```
    ex)
    cd google-map
    ```
1. download node_modules.
    ```
    npm ci
    or
    yarn install
    ```
1. execute run command.
    ```
    npm run serve
    or
    yarn serve
    ```

## commands
### create project
```
vue create (projectname)
```
### uninstall module
```
yarn [global] remove <module>
```

### formatter
***for Mac***
```
Shift + Option + F
```
***for Win***
```
Shift + Alt + F
```
