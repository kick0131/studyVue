# Sample projects

## Usage
1. change into sample project directory.
    ```
    cd google-map
    ```
1. download node_modules.
    ```
    npm ci
    or
    yarn install
    ```
1. execute run command.
    ```
    npm run serve
    or
    yarn serve
    ```

## Project Information

### google-map
Google Map API Sample Project

### mock_study
Simple Vuetify Sample.

### pushReceiver
Receive Firebase Cloud Messaging(Push notification) sample.
